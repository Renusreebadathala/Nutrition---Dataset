# 📌 Project Overview

The **Nutrition Analytics Dashboard** is an end-to-end Business Intelligence project built using Microsoft Power BI to explore nutritional patterns across multiple food categories.

This dashboard enables users to:

✅ Compare Traditional Indian Foods vs Processed Foods  
✅ Analyze nutrient composition across dishes  
✅ Identify high-protein and nutrient-rich foods  
✅ Explore detailed dish-level nutritional insights  
✅ Interact dynamically using slicers, tooltips, and drillthrough pages  

The project focuses on transforming raw nutrition datasets into meaningful business insights using modern data visualization techniques.
